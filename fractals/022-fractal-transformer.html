Fractal Transformer Architecture

This visualization reimagines the transformer neural network architecture using fractal patterns to represent attention mechanisms. In traditional transformers (like GPT, BERT, or Claude), the self-attention mechanism allows each token to attend to every other token through matrix operations. Here, we visualize this as fractal branching structures where information flows through recursive, self-similar patterns.

Each token (represented as a colored dot) enters the network and flows through multiple transformer layers. In each layer, the multi-head attention mechanism splits the processing into parallel attention heads, each forming a unique fractal pattern. These patterns show how tokens interact: connections between tokens represent attention weights, and the fractal structure emerges from the recursive nature of how information propagates through the network.

The fractal branching factor controls how many sub-branches emerge at each level, mimicking how attention heads divide and process information in parallel. The fractal depth represents the hierarchical levels of abstraction - deeper fractals show how transformers build understanding from low-level features to high-level concepts. Different attention patterns (causal, local, sparse) create different fractal topologies, revealing the geometric nature of attention.

Watch how tokens (data particles) flow through the network: they enter at the bottom, get processed through each layer's fractal attention patterns, and emerge transformed at the top. The heatmap shows which tokens attend to which others - bright spots indicate strong attention connections, forming the backbone of the fractal structure.
